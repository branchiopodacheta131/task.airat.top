## 🖥️ What this app does

task.airat.top is a local-first task manager for everyday work and personal lists. It helps you add tasks, sort them, and break larger items into smaller steps. It can also tag tasks for you with AI, so your lists stay neat without much work.

This app is built with React, TypeScript, Vite, and Tailwind CSS. It runs as a web app and is made for fast use on Windows.

## 🧭 How to use task.airat.top

### Add a task

Type your task in the main field and save it. Keep it short and clear, like:

- Pay electric bill
- Clean desk
- Finish report
- Call dentist

### Break a task into steps

For larger work, use task decomposition. This helps turn one big task into smaller items you can finish one at a time.

Example:

- Write blog post
  - Choose topic
  - Draft outline
  - Write first version
  - Edit final copy
  - Publish

### Use AI tags

When AI tagging is turned on, the app can suggest labels for your tasks. This helps group items like:

- Work
- Home
- Errands
- Health
- Urgent
- Personal

### Keep tasks local-first

Local-first means your task data stays close to you on your device. That gives you quick access and keeps your list simple to manage.
